1. Requirement for Shell
	a. The Shell must be able to run these commands:
		/bin/date
                cd /usr
                pwd
                mkdir directory_name
                ls -F
                cat file_name
		% ls -F  > output
                % wc < input > output
                % cat < input > output
		% cat dup.cc | grep main
                % ls | wc
	b. The shell must operate in an infinite loop.
